## ✨ התכונות החדשות

### 1. מייל ברוכים הבאים 👋

- לוגו של SaveDay Events
- הסבר על התכונות של האתר
- כפתור "התחל עכשיו"
- עיצוב בצבעי האתר (ורוד ורוד בהיר)

### 2. איפוס סיסמה משופר 🔐

#### שלב א' - בקשת איפוס
- משתמש מזין את המייל שלו
- המערכת שולחת קוד בן 6 ספרות
- הקוד תקף ל-15 דקות
- המייל מעוצב יפה עם אזהרות ברורות

#### שלב ב' - אישור איפוס
- לאחר איפוס מוצלח, המשתמש מקבל מייל אישור
- המייל כולל טיפים לאבטחה
- כפתור להתחברות מחדש

## 🚀 איך להשתמש?

### בדיקה מהירה

1. הפעל את השרת:
```bash
cd backend
uvicorn main:app --reload
```

2. הרשם במייל אמיתי שלך:
```bash
curl -X POST http://localhost:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "YOUR_EMAIL@gmail.com",
    "password": "password123",
    "full_name": "השם שלך"
  }'
```

3. בדוק את המייל שלך - אמור להגיע מייל מעוצב! 📧

### בדיקת איפוס סיסמה

1. בקש קוד איפוס:
```bash
curl -X POST http://localhost:8000/api/auth/forgot-password \
  -H "Content-Type: application/json" \
  -d '{"email": "YOUR_EMAIL@gmail.com"}'
```

2. קבל קוד במייל
3. אפס את הסיסמה:
```bash
curl -X POST http://localhost:8000/api/auth/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "YOUR_EMAIL@gmail.com",
    "code": "123456",
    "new_password": "newpassword123"
  }'
```

4. תקבל מייל אישור!
